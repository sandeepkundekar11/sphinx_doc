1. Introduction¶

This documentation provides a detailed guide for front-end development, UI/UX design using HTML, CSS, JavaScript, Wireframe Tool, Tailwind CSS, and template selection. The tasks are outlined as follows:

  1. Find a wireframe tool (Figma) compatible with both Linux and Windows.

  2. Learn and experiment with Tailwind CSS.

  3. Find a suitable template for the website design.

  4. Create a website mapping.
